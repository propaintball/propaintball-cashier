📡 NFC / RFID Čítačka — technická analýza
Akasa čítačka (Mifare Classic 1K) v HID Keyboard móde.
Toto je analýza 4 možností ako pripojiť NFC čítačku:
✅ ODPORÚČANÉ: HID Keyboard Emulation
Čítačka sa správa ako klávesnica — UID náramku odošle ako text do aktívneho inputu. Funguje bez ovládačov, bez softvéru.
Postup: 1. Klikni do poľa "ID náramku" 2. Prilož náramok → UID sa automaticky vloží.
🔵 Web Serial API (Chrome/Edge)
Priame čítanie cez USB Serial port. Vyžaduje Chrome/Edge na PC/Mac, nefunguje na Safari/iPad.
⚪ Web HID API
Priame HID komunikácia. Niektoré čítačky vyžadujú vlastný protokol. Komplexná implementácia.
⚪ Lokálny proxy server (Node.js)
Node.js server na PC číta NFC a posiela WebSocket správy. Vyžaduje inštaláciu na počítači.
💡 Pre Akasa + Mac Mini M4: Pripoj čítačku cez USB-C. macOS rozpozná HID zariadenie automaticky. V Pokladni prepni na NFC/RFID tab → klikni do poľa UID → prilož náramok. Funguje v Safari aj Chrome bez ovládačov.

🔗 Windows PC + Akasa → iPad Bridge
Tvoja konfigurácia: Akasa USB-C → Windows PC, iPad na rovnakej WiFi

Postup (raz nastaviť):
1. Stiahni Python script nižšie na Windows PC
2. Nainštaluj Python: python.org/downloads
3. V CMD zadaj: pip install websockets keyboard
4. Spusti script: python nfc_bridge.py
5. Script vypíše IP adresu PC → zadaj ju nižšie na iPade
6. Klikni Pripojiť → prikladanie náramku automaticky priradí hráča
